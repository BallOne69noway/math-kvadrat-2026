<a name="english"></a>
## 🇺🇸 English
**Math Kvadrat** is an interactive system for hosting real-time mathematics competitions. The project is designed to automate answer collection and display results instantly.

### 🚀 Key Features
* **Interactive Interface**: User-friendly form for participants to submit answers.
* **Realtime Database**: Instant data synchronization via Google Firebase.
* **Automatic Scoring**: The system records response time and accuracy.
* **Secure Hosting**: Deployed on Firebase Hosting.

### 🛠 Technologies
* **Frontend**: HTML5, CSS3, JavaScript.
* **Backend/DB**: Google Firebase (Realtime Database & Hosting).
* **Version Control**: Git & GitHub.

## 📦 Local Setup / Локальная установка
1. **Clone repo**: 
   `git clone https://github.com/BallOne69noway/math-kvadrat-2026.git`
2. **Install Firebase CLI**: 
   `npm install -g firebase-tools`
3. **Run local server**: 
   `firebase serve`
