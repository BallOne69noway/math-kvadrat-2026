# 🧮 Math Kvadrat 2026

**Math Kvadrat** — это интерактивная система для проведения математических олимпиад в реальном времени. Проект разработан для автоматизации сбора ответов и мгновенного отображения результатов.

## 🚀 Основные функции
* **Интерактивный интерфейс**: Удобная форма для ввода ответов участниками.
* **Realtime Database**: Мгновенная синхронизация данных через Google Firebase.
* **Автоматический подсчет**: Система фиксирует время и точность ответов.
* **Безопасный хостинг**: Развернуто на Firebase Hosting.

## 🛠 Технологии
* **Frontend**: HTML5, CSS3, JavaScript.
* **Backend/DB**: Google Firebase (Realtime Database & Hosting).
* **Version Control**: Git & GitHub.

## 📦 Как развернуть проект локально
1. Клонируйте репозиторий:
   ```bash
   git clone [https://github.com/BallOne69noway/math-kvadrat-2026.git](https://github.com/BallOne69noway/math-kvadrat-2026.git)
2.Установите Firebase CLI (если не установлен):
npm install -g firebase-tools
3.Запустите локальный сервер:
firebase serve
